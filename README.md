# odin-recipes.

This is my first project from **The Odin Project Foundations** course.

The goal of this project is to build a basic recipe website using only HTML. The website contains a main page with links to multiple recipe pages.

By completing this project, I demonstrated the following skills:

- Creating basic HTML pages
- Using headings, paragraphs, images, and lists
- Creating links between multiple pages
- Organizing project files and folders
- Using Git and GitHub to manage and track a project
